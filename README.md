## Echo-intro 

本项目为 Golang Echo 框架官方文档的汉化文档  

### 使用

1. 网页使用hugo运行，如果没有安装hugo，请前去安装[hugo](https://gohugo.io/)  
2. 在`website-zh`目录运行 `./hugo serve`  
3. 在浏览器访问`loalhost:1313`即可打开项目

### 参与翻译

fork 该项目，修改后提交pr即可。  
汉化版文件在`website-zh`目录里，`content` 目录下的文件为网站内容，均为 Markdown 文件。
